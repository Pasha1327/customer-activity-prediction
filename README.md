# Предсказание покупательской активности

Модель бинарной классификации для предсказания вероятности снижения покупательской активности клиентов интернет-магазина в следующие 3 месяца. На основе предсказаний — сегментация и персонализированные бизнес-рекомендации.

## Результат

**LogisticRegression (L1)** — ROC-AUC **0.91** на кросс-валидации, **0.89** на тестовой выборке.

Топ-4 важных признака по SHAP: `страниц_за_визит` (0.55), `средний_просмотр_категорий_за_визит` (0.59), `минуты_предыдущий_месяц` (0.57), `минуты_текущий_месяц` (0.42).

## Стек

Python · scikit-learn · pandas · seaborn · SHAP · phik · Jupyter

## Данные

4 таблицы интернет-магазина «В один клик» (1300 клиентов):
- `market_file` — поведение на сайте: маркетинговые коммуникации, акционные покупки, категории, ошибки сервиса
- `market_money` — выручка за 3 периода (текущий / предыдущий / препредыдущий месяц)
- `market_time` — время на сайте в минутах за 2 периода
- `money` — среднемесячная прибыль клиента

## Что сделано

**Предобработка:**
- Исправлены опечатки в категориальных признаках (`стандартт` → `стандарт`, `предыдцщий_месяц` → `предыдущий_месяц`)
- Удалён клиент с аномальной выручкой (106k при среднем 5k)
- Удалены 3 клиента с нулевой выручкой за все периоды
- Таблицы с временными рядами преобразованы через `pivot_table` и объединены через `merge`

**Моделирование:**
- Сравнение 4 моделей через `RandomizedSearchCV` + `Pipeline`: KNN, DecisionTree, LogisticRegression, SVC
- `ColumnTransformer`: OHE для бинарных признаков, `OrdinalEncoder` для типа сервиса, `MinMaxScaler` для числовых
- Корреляционный анализ через phik-матрицу (смешанные типы данных)
- Мультиколлинеарность между признаками не обнаружена

**Интерпретация:**
- SHAP bar plot и beeswarm plot
- Признаки с наименьшим влиянием: `маркет_актив_тек_мес`, `ошибка_сервиса`, `популярная_категория_Товары_для_детей`

**Сегментация и бизнес-выводы:**
- Выделены сегменты по топ-категориям: «Товары для детей» (329 клиентов) и «Домашний текстиль» (250 клиентов)
- В общей выборке активность снижается у 38% клиентов; в сегменте «Товары для детей» — у 44%, «Домашний текстиль» — у 40%
- Рекомендации: сезонные распродажи, бонусные программы, скидки на наборы товаров
